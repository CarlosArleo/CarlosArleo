deterministic boundaries that an agent cannot cross without ceasing to operate.

**Constitutional Halt**: a designed, safe transfer of authority back to human
operators, with a complete deterministic audit log. This is the opposite of
Silent Failure.

This is not theoretical. It is deployed and empirically validated.
